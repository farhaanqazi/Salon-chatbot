# Salon-chatbot

### 💇‍♀️ AI-Powered Beauty Parlour & Salon Dashboard
A complete end-to-end management solution designed to modernize beauty parlours and salons. 

**Key Features:**
- **AI Chatbot Integration:** Automated, intelligent customer interactions for booking appointments via channels like WhatsApp and Telegram.
- **Real-Time Dashboard:** A premium, "Dark Craft" inspired analytical dashboard providing live KPI tracking, revenue charting, and appointment monitoring.
- **Modern Tech Stack:** 
  - **Frontend:** React 19, Vite, Tailwind CSS, Zustand, and TanStack Query.
  - **Backend:** Python FastAPI, SQLAlchemy, and LLM integrations.
  - **Database & Auth:** Secure session management and PostgreSQL tables powered by Supabase.
